# Vulnerabilities

Vulnerability is a weakness in a system that can be exploited.

--------------------------------

## Types of Vulnerabilities

Software bugs  
Weak passwords  
Outdated software  
Misconfigurations  
Open ports  

--------------------------------

## Software Bugs

Errors in code

Example
Login bypass bug

--------------------------------

## Weak Passwords

Easy to guess passwords

Examples
123456
password
admin

--------------------------------

## Outdated Software

Old versions with known exploits

Example
Old Apache server vulnerable

--------------------------------

## Misconfiguration

Incorrect settings

Examples
Open S3 bucket  
Public database  
Default credentials

--------------------------------

## Open Ports

Unnecessary services running

Example
Port 21 FTP open
Attacker brute forces login

--------------------------------

## Vulnerability Flow

Vulnerability → Exploit → Access
