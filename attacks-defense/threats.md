# Threats

Threat is anything that can harm a system.

--------------------------------

## Types of Threats

Malware  
Phishing  
DDoS  
Insider Threat  
Password Attacks  

--------------------------------

## Malware

Malicious software

Types
Virus  
Worm  
Trojan  
Ransomware  

Example
File infected → system compromised

--------------------------------

## Phishing

Fake email or website

Example
Fake login page  
User enters password  
Attacker gets credentials

--------------------------------

## DDoS

Distributed Denial of Service

Many systems flood server

Result
Website goes down

--------------------------------

## Insider Threat

Employee misusing access

Example
Copy company database

--------------------------------

## Password Attacks

Brute force  
Dictionary attack  
Credential stuffing
