# CIA Triad

CIA stands for:

Confidentiality  
Integrity  
Availability  

--------------------------------

## Confidentiality

Only authorized users can access data

Examples
- Password protection
- Encryption
- Access control

Attack
Data leak

--------------------------------

## Integrity

Data should not be modified

Examples
- Hashing
- Checksums
- File permissions

Attack
Data tampering

--------------------------------

## Availability

System should be available

Examples
- Redundant servers
- Backups
- Load balancing

Attack
DDoS attack
