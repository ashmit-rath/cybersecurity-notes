# Router vs Switch

## Switch
- Connects devices inside LAN
- Uses MAC addresses
- Works on Layer 2
- Faster inside local network

Example:
PC → Switch → PC

## Router
- Connects different networks
- Uses IP addresses
- Works on Layer 3
- Connects LAN to Internet

Example:
PC → Router → Internet

## Gateway
- Router acts as gateway
- Used to reach outside network

Example:
192.168.1.1 → default gateway
