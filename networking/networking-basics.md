# Networking Basics

## IP Address
- IP address is divided into 4 octets
- Range: 0-255
- Example: 192.168.1.1

## MAC Address
- Unique hardware address
- Example: a4:c3:f0:85:ac:2d
- First half = vendor
- Second half = unique device

## ICMP
- Internet Control Message Protocol
- Used for ping

## Spoofing
- Pretending to be another device
- Example: IP spoofing, MAC spoofing
