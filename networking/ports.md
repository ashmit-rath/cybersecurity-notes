# Ports

## What is a Port
- Port identifies a specific service on a system
- Works with IP address
- Format: IP:PORT
Example: 192.168.1.1:80

## Common Ports

21  - FTP  
22  - SSH  
23  - Telnet  
25  - SMTP  
53  - DNS  
80  - HTTP  
110 - POP3  
139 - NetBIOS  
143 - IMAP  
443 - HTTPS  
445 - SMB  
3389 - RDP  

## Port Types

Well Known Ports
- 0 - 1023

Registered Ports
- 1024 - 49151

Dynamic Ports
- 49152 - 65535
