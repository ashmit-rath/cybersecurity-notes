# Network Topologies

## Star Topology
Advantages:
- Easy to add devices
- Easy troubleshooting

Disadvantages:
- More cables required
- Central device dependency

## Bus Topology
Advantages:
- Cost efficient
- Simple setup

Disadvantages:
- Difficult troubleshooting
- Single cable failure affects network

## Ring Topology
Advantages:
- One direction data flow

Disadvantages:
- One cable failure breaks whole network
