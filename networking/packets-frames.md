# Packets and Frames

## Packet
- Data is broken into smaller pieces called packets
- Packets travel across the network
- Each packet contains:
  - Source IP
  - Destination IP
  - Data

## Frame
- Frame is packet wrapped with MAC addresses
- Used in local network communication
- Contains:
  - Source MAC
  - Destination MAC
  - Data

## Encapsulation
- Data wrapped layer by layer
- Application → Transport → Network → Data Link

## Why packets are used
- Efficient transmission
- Error handling
- Routing flexibility
