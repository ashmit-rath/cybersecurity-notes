# Windows Basics

## Windows File System
Windows uses drives like:
- C:\
- D:\
- E:\

Folders example:
C:\Users\Username\Desktop

## Important Directories
- C:\Windows
- C:\Program Files
- C:\Users
- C:\Temp

## Useful Commands
- dir → list files
- cd → change directory
- cls → clear screen
- mkdir → create folder
- rmdir → remove folder
- copy → copy files
- move → move files

## Hidden Files
Use:
dir /a

## Recursive Search
Use:
dir /s
