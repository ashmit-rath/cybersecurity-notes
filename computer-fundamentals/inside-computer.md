# Inside a Computer System

## Main Components
CPU - processes instructions  
RAM - temporary memory  
Storage - permanent data (HDD/SSD)  
Motherboard - connects components  
Power Supply - provides power  

## CPU Parts
ALU - calculations  
Control Unit - manages instructions  
Registers - very fast memory  

## RAM
Temporary memory  
Data lost after shutdown  

## Storage
HDD - slower, cheaper  
SSD - faster, expensive
