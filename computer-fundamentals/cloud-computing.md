# Cloud Computing Fundamentals

Cloud computing means using internet-based servers instead of local computer.

Example  
Google Drive  
AWS  
Azure  

## Types of Cloud

Public Cloud  
Shared infrastructure (AWS, Azure)

Private Cloud  
Used by single organization

Hybrid Cloud  
Combination of both

## Cloud Services

IaaS – Infrastructure (VM, storage)  
PaaS – Platform (runtime environment)  
SaaS – Software (Gmail, Google Docs)

## Advantages

No hardware required  
Scalable  
Access from anywhere  
Cost efficient
