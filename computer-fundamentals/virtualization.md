# Virtualization

Virtualization allows running multiple systems on one machine.

Example  
One laptop → Windows + Linux VM  

## Hypervisor
Software that creates virtual machines

Types  
Type 1 – Bare metal (VMware ESXi)  
Type 2 – Hosted (VirtualBox, VMware)

## Why Virtualization
Isolation  
Testing malware safely  
Run multiple OS  
Lab environments  

## Example
Host OS: Windows  
Guest OS: Kali Linux  
Software: VirtualBox
