# Data Representation

## Binary
Computers store data in binary (0 and 1)

Example
Decimal: 5
Binary: 101

## Hexadecimal
Used for compact representation

Example
Hex: 0x1A
Decimal: 26

## ASCII
Text to number mapping

Example
A = 65
a = 97

## Bits and Bytes
1 bit = 0 or 1  
8 bits = 1 byte  

Example
01000001 = A
