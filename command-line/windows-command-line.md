# Windows Command Line (CMD)

Windows Command Line allows users to interact with Windows using text commands.

---

# System Information

## ver
Shows Windows version

Example:
ver

## systeminfo
Displays system information

Example:
systeminfo

## set
Shows environment variables

Example:
set

---

# Network Commands

## ipconfig
Shows IP address

Example:
ipconfig

## ipconfig /all
Shows full network details

Example:
ipconfig /all

## ping
Check connectivity

Example:
ping google.com

## tracert
Trace route to destination

Example:
tracert google.com

## nslookup
Find domain IP

Example:
nslookup google.com

## netstat
Shows active connections

Example:
netstat

## netstat -abon
Shows ports + processes

Example:
netstat -abon

---

# Directory Navigation

## cd
Change directory

Example:
cd Desktop

## dir
List files

Example:
dir

## dir /a
Show hidden files

Example:
dir /a

## tree
Shows folder structure

Example:
tree

---

# File Operations

## mkdir
Create folder

Example:
mkdir test

## rmdir
Remove folder

Example:
rmdir test

## copy
Copy file

Example:
copy file.txt D:\backup

## move
Move file

Example:
move file.txt Desktop

## del
Delete file

Example:
del file.txt

## type
Read file

Example:
type file.txt

---

# Process Commands

## tasklist
Shows running processes

Example:
tasklist

## taskkill
Kill process

Example:
taskkill /PID 1234

---

# Disk Commands

## chkdsk
Check disk errors

Example:
chkdsk

## sfc /scannow
Scan system files

Example:
sfc /scannow

---

# System Commands

## shutdown /s
Shutdown PC

## shutdown /r
Restart PC
