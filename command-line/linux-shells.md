# Linux Shells

Shell allows users to interact with Linux OS.

Common shells:
- bash
- sh
- zsh
- fish

---

# What Shell Does

- Executes commands
- Manages processes
- File operations
- scripting

---

# Basic Shell Commands

## echo
Print text

example:
echo hello

---

## whoami
Shows current user

example:
whoami

---

## pwd
Print working directory

example:
pwd

---

## ls
List files

example:
ls

---

## cd
Change directory

example:
cd Desktop

---

# Switching Shell

Example:
bash
zsh
sh

---

# History

Shows command history

example:
history

---

# Piping

Pipe output

symbol:
|

example:
ls | grep test

---

# Shell Differences

bash → default linux shell  
zsh → advanced features  
sh → basic shell  
fish → user friendly
