# DNS (Domain Name System)

## What is DNS
- DNS converts domain name to IP address
- Example: google.com → IP

## Why DNS is used
- Humans remember names
- Computers use IP addresses

## DNS Lookup Process
1. Browser cache check
2. OS cache check
3. Resolver query
4. Root server
5. TLD server
6. Authoritative server
7. IP returned

## Record Types
A - IPv4
AAAA - IPv6
CNAME - Alias
MX - Mail server
TXT - Text record
