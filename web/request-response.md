# Request and Response

## Request
- Sent by client (browser)
- Contains:
  - Method (GET, POST)
  - URL
  - Headers
  - Body (optional)

Example:
GET /index.html

## Response
- Sent by server
- Contains:
  - Status code
  - Headers
  - Body

Example:
200 OK

## Common Status Codes

200 - OK  
301 - Redirect  
403 - Forbidden  
404 - Not Found  
500 - Internal Server Error  

## Flow

Client → Request → Server  
Server → Response → Client
